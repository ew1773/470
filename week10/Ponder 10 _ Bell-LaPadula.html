Ponder 10 : Bell-LaPadula

Due Saturday at 5:00 PM MST

It is often necessary for a computer system to provide Confidentiality assurances to a data store when more than one type of user requires access to the system. Your assignment is to retrofit an existing program by implementing the Bell-LaPadula. The following data flow diagram describes the current operation of the system:

The current grades program gives any user unlimited authority to change anyone's grades. With the changes you will make, you will have three classes of users:

  • Bob, Hans: can view any grade
  • Sam, Sue, Sly: can only view their own grades
  • Everyone else: cannot view any grade

To get full credit for this assignment, please:

  • Implement the Bell-LaPadula data structure in your code. Make sure these data structures work the way it is described in the text.
  • Make sure that resource allocation decisions are made at the "best possible location" to provide confidentiality and integrity assurances. This is probably best done at the StudentGrade part of the program.
  • Make it "easy and intuitive" to change policy settings. This means adding new users or groups should not require any modification of the logic of the program. For this assignment, it is OK to put the policy settings in constant data structures.
